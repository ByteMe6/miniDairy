# miniDairy

Минималистичный дневник на день — пишешь что угодно, записи исчезают через 24 часа. Удобно для планов, напоминаний или заметок, которые не нужно хранить.

[English](./README.md)

## Возможности

- Вход через Google и email/пароль (Firebase Auth)
- Написание и удаление постов
- Посты сортируются по новизне
- Автоматическая очистка через 24 часа
- Toast-уведомления

## Технологии

- Vanilla JavaScript
- Firebase Auth + Realtime Database
- Toastify
- nanoid

## Запуск

Бандлер не нужен — открой `index.html` напрямую или запусти любой статический сервер.

Firebase-конфиг уже встроен. Чтобы использовать свой проект, замени конфиг в `js/index.js`.
